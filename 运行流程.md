# Spider_XHS 运行流程

## 1. 环境准备

- Python 3.10+
- Node.js 20+
- Chrome、Edge 或 Chromium 任选一个

## 2. 安装依赖

```bash
pip install -r requirements.txt
npm install
```

## 3. 启动 Web 控制台

```bash
python3 web_app.py
```

浏览器打开：

```text
http://127.0.0.1:8765
```

## 4. 登录并开始采集

推荐直接走 Web 控制台：

1. 打开“设置”
2. 点击“打开浏览器登录”
3. 在弹出的浏览器里登录小红书
4. 等待系统自动保存 Cookie
5. 配置关键词、采集数量和筛选条件
6. 返回首页开始采集

默认输出目录：

```text
datas/markdown_datas/<运行批次>/<关键词>/<笔记标题>/
```

## 5. 可选：脚本方式运行

如果你只是想跑示例脚本：

```bash
python main.py
```

注意：

- `main.py` 里的示例链接可能会过期
- 真正运行前通常要换成你自己的参数
- 没有有效 Cookie 时请求会失败

## 6. 最短启动方式

如果只是想尽快跑起来：

```bash
python3 web_app.py
```

然后打开 `http://127.0.0.1:8765`，在设置页完成登录后即可开始采集。
